# Motiontale Pro (demo)

Marketing + early-access flow scaffold for a motion-control product.

## Stack

- Next.js (App Router)
- Tailwind CSS
- next-auth (Credentials provider demo)

## Getting started

```bash
pnpm install
pnpm dev
```

Open: http://localhost:3000
